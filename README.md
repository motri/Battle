# Battle
This is a basic single player battle game.

### Setup
```
Clone this repo.
Run bundle install.
In the command line run "bundle exec rackup -p 3000".
Navigate to "localhost:3000" in your browser.
```

### Tech
* Ruby.
* Rspec for testing.
* Sinatra for simple web interaction.
